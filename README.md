# Vue.js AI Agent (Dashboard)

一個基於 Vue.js 打造的 AI Agent 思考歷程與狀態視覺化儀表板。

## 🚀 專案線上 Demo

**網頁網址：** [https://vue-ai-agent-chi.vercel.app/](https://vue-ai-agent-chi.vercel.app/)

---

## 🛠️ 技術棧與架構亮點

- **前端核心**：使用 **Vue.js 3** 搭配 **Vite** 構建工具，確保極速的開發熱更新（HMR）與高效能的生產環境打包（Build）。
- **元件與 UI 架構**：採用模組化元件設計，將 AI Agent 的不同思考階段、日誌輸出與狀態看板進行解耦，提升程式碼的可維護性與擴充性。
- **響應式設計 (RWD)**：優化多端裝置的流暢體驗，確保在不同螢幕尺寸下皆能完美呈現。
- **自動化部署 (CI/CD)**：整合 **Vercel 雲端託管平台** 與 **GitHub** 儲存庫。配置現代化網頁部署流程，實現本地代碼一經 Git Push，雲端即時自動編譯並更新上線。
